# About This Page

This page loads public articles from Markdown files stored in the `articles/` directory.

To publish a new article:

- Add a new Markdown file to `articles/`.
- Add an entry for it in `articles/index.json`.
- Open `articles.html?article=your-slug` to check it.

The `slug` in `index.json` should match the Markdown filename without `.md` unless a custom `file` field is provided.
